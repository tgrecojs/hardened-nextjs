# hardened-nextjs
Next.js App with Hardened JS enabled.

### Instructions
```sh
git clone https://github.com/tgrecojs/hardened-nextjs && cd hardened-nextjs
# install deps with npm
~/hardened-nextjs npm install
# run dev env
~/hardened-nextjs npm run dev
```
